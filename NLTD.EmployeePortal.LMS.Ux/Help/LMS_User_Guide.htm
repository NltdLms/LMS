About LMS

LMS is intranet website which is accessible with in TCC Corp network. LMS is website helps to manage (Apply, Approve, Reports) different requests.

Website URL: http://NLTI60:9002

Functional Overview

·         Configurable Holiday Office for an Employee

·         Configurable Weekly Off days

·         Configurable Time based, Day based request types

·         Automatic Leave Balance Credit/Debit for actions on Leaves

·         Hierarchy based data fetching on demand

·         View Request History (Data will be fetched based on the appropriate menu link section)  To see submitted requests by self, team and all based on roles.

·         Day wise Report (Data will be fetched based on the appropriate menu link section)  To see requests with in specific date range. This can be used to see how many are/will be off on particular day. Clicking on Export to Excel fills data based on Search Criteria and download Excel.

·         Year wise Report (Data will be fetched based on the appropriate menu link section)  To see leave summary for the selected year for self, team and all based on roles. Export to Excel option available.

·         Permissions Report (Data will be fetched based on the appropriate menu link section)  To see time based requests for self, team and all based on roles. Export to Excel option available.

If new request is applied, the status will be in pending.

If the request is CL/SL or PL, the Applied leave count will be incremented by applied no. of days and Balance leaves will be reduced by same number.

If request is LWP, CO, or WFH, Applied leave count will be incremented by applied no. of days and there will not be any change in Balance Leaves.

If leave request is Approved, the Applied count will be reduced that many no. of days and Used Leaves will be incremented by same number.

If leave request is rejected, the Applied count will be reduced that many no. of days. Balance Leaves will be incremented by same number for CL/SL and PL.

If leave request is Cancelled after approval of leave, Used leaves will be reduced by the no. of days in leave request and Balance leaves will be incremented by same number.

How to apply a new Request

1.       Click My Request àApply New Request

2.       Clicking on View Leave Summary shows the leave summary for current year.

3.       Steps to apply:

a.       Select appropriate option from Request Type.

b.      Select From, Part of Day, To and Part of Day from respective choices.

c.       Enter Reason in the Reason box. Click Apply button.

d.      Click on Confirm

e.       Confirmation message about Request submission will be shown.

 

 

How to see submitted Request Status

1.       Click My Requests -> Request History

a.       By default, todays date will be selected in Date Range.

2.       Click to see pre-defined date ranges or click on Custom Range to open Calendar to select From and To Date.

3.       Click Search.

a.       The screen shows requests with in the specified date range.

 

4.       Click on Name or + sign to expand and show the contents inside the box.

 

Approve/Reject a Request

1.       Click Manage Team Leaves -> Manage. This option is available only with Team Management Role in LMS.

2.       The screen by default populates names of team members who has applied a request.

3.       Clicking on the name, expands and shows the complete request detail.

4.       Enter Approver Comments. Click Approve or Reject.

5.       Once rejected, the request status cannot be altered any more. User can apply new request same as rejected one.

6.      Approved request can be cancelled if user didnt utilize the request.

 

Cancel the approved request

1.       In the below Manage screen, select check box Show Approved Requests. From Date and To Date can be entered to narrow down the search result.

2.       Click Search.

3.       The screen populates names of team members with Approved Requests.

4.       Click on Name of team member name to full details of Approved Requests.

5.       Enter Approver Comments and click Cancel Request.

6.      On success, a message will be displayed. Leave balance will be adjusted between Used and Balance Leaves for applicable leaves.
